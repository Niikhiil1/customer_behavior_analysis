# customer_behavior_analysis
Data analysis project showcasing customer behavior analysis using sql, python and power bi.


 Data Analytics Project – End-to-End Analysis 

 Project Overview 

This project demonstrates an end-to-end **data analytics workflow** starting from raw data to interactive dashboard visualization. The goal of the project is to extract meaningful insights from the dataset using Python, SQL, and Power BI.

The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, and the creation of an interactive dashboard for better business understanding.


Project Workflow 

 1. Data Loading 

* The dataset was loaded and processed using **Python**.
* Libraries such as **Pandas** and **NumPy** were used for data manipulation and initial inspection.

 2. Exploratory Data Analysis (EDA) 

* Performed exploratory analysis to understand:

  * Data distribution
  * Patterns and trends
  * Relationships between variables
* Identified missing values, duplicates, and outliers.

 3. Data Cleaning 

* Handled missing values and inconsistent data.
* Removed duplicate records.
* Standardized and formatted columns for better analysis.

 4. SQL Analysis 

* Imported the cleaned dataset into a SQL database.
* Wrote SQL queries to analyze:

  * Key metrics
  * Customer behavior
  * Sales or performance trends
* Used SQL concepts such as **joins, aggregations, filtering, and window functions**.

 5. Dashboard Development 

* Built an **interactive dashboard using Power BI**.
* The dashboard provides clear insights through:

  * KPIs
  * Trend analysis
  * Category-wise performance
  * Interactive filters and visualizations.

 Tools & Technologies

 Key Outcomes
